# 🚀 DevInsight Server

Backend API for **DevInsight**, a real-time API monitoring + GitHub code analysis platform.

It handles:
- API metrics tracking
- GitHub repository analysis
- Code issue detection
- Analysis history storage
- Performance monitoring

---

## 🧠 Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- Axios
- Passport.js (Auth)
- Express Session
